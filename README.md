Projeto Front-End – Delícias da Tia Nokinha




O sistema “Delícias da Tia Nokinha” é um site institucional desenvolvido para divulgação de cursos de confeitaria online e presenciais.

A plataforma apresenta informações sobre a história da confeiteira, cursos oferecidos, formas de contato e direcionamento direto para WhatsApp e Instagram.

O objetivo do projeto é criar uma interface moderna, responsiva e interativa, permitindo que clientes conheçam os cursos disponíveis e entrem em contato de forma simples e rápida.
