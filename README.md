Projeto Final Front-End – Delícias da Tia Nokinha


Descrição do Sistema (Mini-Mundo)

O sistema “Delícias da Tia Nokinha” é um site institucional desenvolvido para divulgação de cursos de confeitaria online e presenciais.

A plataforma apresenta informações sobre a história da confeiteira, cursos oferecidos, formas de contato e direcionamento direto para WhatsApp e Instagram.

O objetivo do projeto é criar uma interface moderna, responsiva e interativa, permitindo que clientes conheçam os cursos disponíveis e entrem em contato de forma simples e rápida.


Tecnologias e Funcionalidades

Tecnologias Utilizadas:

HTML5
CSS3
Java Script

2.  Funcionalidades Implementadas:

Estrutura semântica organizada

Seção de FAQ (Dúvidas Frequentes) interativa

Procedimentos de Execução Local

Clonar o repositório ou baixar o arquivo .zip.
Acessar a pasta principal do projeto.
Abrir o arquivo index.html no navegador.
O sistema não necessita de servidor ou dependências externas para funcionamento básico.
